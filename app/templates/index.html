Project Details

This project utilizes a neural network to recognize both handwritten letters and digits. The model is trained on the EMNIST dataset, which includes a variety of characters. The neural network architecture involves convolutional and dense layers to efficiently learn patterns from the input images.

The training process is enhanced by callbacks such as ModelCheckpoint, EarlyStopping, and ReduceLROnPlateau. These callbacks ensure the model's performance is optimized during training.

After training, the model is evaluated on a test dataset to assess its generalization performance. Additionally, the project demonstrates predictions on random samples from the test data, showcasing the model's ability to recognize handwritten characters.
